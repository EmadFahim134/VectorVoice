
# VectorVoice

VectorVoice is a text-to-speech software that is used as an concept for using Pandas & Numpy to create a simple tts software using math and pure language.


## Requirments
For the Requirments to run this project you need:
- Uv
- Git
- Pyhton 3.14.3
## Installation

To run the project:

- Clone this project via git:
```bash
  git clone https://github.com/EmadFahim134/VectorVoice.git
```
- Run:
```python
  uv run main.py
```
